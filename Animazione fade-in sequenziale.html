Perfetto 👍 Ti preparo un esempio semplice in HTML + CSS (puoi aggiungere anche un po’ di JavaScript per la sequenza).

L’idea è:

  • il logo (logo.png) appare con un fade-in di 1.5 secondi;

  • quando il logo ha finito la sua animazione, parte il fade-in del testo sotto.

Ecco il codice:

html
<!DOCTYPE html>
<html lang="it">
<head>
  <meta charset="UTF-8">
  <title>Fade In Sequenziale</title>
  <style>
    body {
      display: flex;
      flex-direction: column;
      align-items: center;
      justify-content: center;
      height: 100vh;
      background: #f5f5f5;
      margin: 0;
    }

    img, .testo {
      opacity: 0;
    }

    .fade-in {
      animation: fadeIn 1.5s forwards;
    }

    @keyframes fadeIn {
      from { opacity: 0; }
      to { opacity: 1; }
    }
  </style>
</head>
<body>

  <img id="logo" src="logo.png" alt="Logo" width="200">
  <h1 id="scritta" class="testo">Benvenuto sul sito!</h1>

  <script>
    // Primo fade-in del logo
    const logo = document.getElementById("logo");
    const scritta = document.getElementById("scritta");

    // parte l'animazione appena caricata la pagina
    window.addEventListener("load", () => {
      logo.classList.add("fade-in");

      // dopo 1.5s (durata del fade-in del logo), parte il fade-in della scritta
      setTimeout(() => {
        scritta.classList.add("fade-in");
      }, 1500);
    });
  </script>

</body>
</html>

🔹 Funzionamento:

  • opacity: 0 iniziale → invisibile.

  • .fade-in → attiva l’animazione di 1.5s.

  • Con setTimeout faccio partire il secondo fade-in solo dopo che il primo è finito.
